# SunRise
무.려. AI를 활용해서 제작된 해돋이용 패브릭 모드 입니다.
### 실행 환경

* Java 21
* 패브릭 1.21.11

### 적용법

[Releases](https://github.com/1cowoo/SunRise/releases) 페이지에서 최신 버전 모드(sources 없는거) 받고서 서버의 `mods/` 폴더에 옮기고 서버를 재시작 합니다.


### 수정하고 싶은 분들이 참고해야 할 점
1. 코드는 첫 플러그인 개발이라 수정이 쉽지않게 개떡같이 돼있을겁니다.(주석도 거의 안달려있음)
2. 코틀린 아니고 걍 자바언어 입니다.

### 빌드 방법
```html
git clone https://github.com/1COWOO/SunRise
cd SunRise
./gradlew build
cd build/libs (자동 실행이 안될수있으니 따로 복사해두세요)
```
### 주의점 
유튜브 같은 플랫폼에서 사용할시 이 깃허브 링크 혹은 [유튜브 채널](https://youtube.com/@KOWOO코우)을 영상 설명창에 써주세요.
